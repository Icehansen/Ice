“Sexual harassment and assault is a very big issue in our society, but no one talks about it,” Bishops said.

PROJECT 3

The third project is an exam project. The third project is about creating a process portfolio and an HTML website. This is an individual task as you have to create and code the website yourself.






Process


It all started with the fact that I chose to sketch how my website should look with colors and text types, what target group it is for. Next, I have chosen to make wireframes inside Adobe XD how the website should look. Since I have designed wireframes on Adobe XD, I have created a mood board and style tile.
Mood-board I have designed to find a concept and which themes to go with. Style tile provided an overview of which font to use, which buttons to include and which colors to use for the website. I have therefore chosen that my theme should be minimalist and simple, to get a quiet and calm mood.

PROJECT 2

The next project of the semester was Installation. The group was to find and develop an installation concept for H. C. Andersen’s museum in Odense.

PROJECT 1

The first project was for the group to re-design fyensrideklub.dk. Each group got a website to work with. The task was, that the group had to redesign the “fyensrideklub.dk” website, so it is more manageable, more modern and more credible.



Additionally, the group had to design a client report, to tell and give a description of what the idea was and what the outcome would look like. Based on the result, the group had used various methods to redesign the website, eg the NUG model to give an insight into the various problems on the website, besides, the group had used the C.R.A.P model to give the group an insight into how user-friendly the website is. Eventually, the group had used the SMUK model to find the right target group to focus on.
